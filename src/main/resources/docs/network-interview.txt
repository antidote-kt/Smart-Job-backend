计算机网络面试题：

1. OSI七层模型？
- 物理层：传输比特流
- 数据链路层：帧传输和错误检测
- 网络层：路由选择和IP寻址
- 传输层：端到端通信（TCP/UDP）
- 会话层：建立和管理会话
- 表示层：数据加密和压缩
- 应用层：为应用程序提供网络服务

2. TCP和UDP的区别？
TCP（传输控制协议）：
- 面向连接
- 可靠传输
- 流量控制
- 拥塞控制
- 全双工通信

UDP（用户数据报协议）：
- 无连接
- 不可靠传输
- 开销小
- 速度快

3. HTTP和HTTPS的区别？
- HTTP：明文传输，端口80
- HTTPS：加密传输（SSL/TLS），端口443，提供身份验证

4. 什么是DNS？
域名系统（Domain Name System），将域名转换为IP地址的分布式数据库系统。

5. 三次握手和四次挥手？
三次握手建立连接：SYN → SYN+ACK → ACK
四次挥手断开连接：FIN → ACK → FIN → ACK
